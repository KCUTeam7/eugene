can use flexbox and grid together
flexbox: 1d layout module
grid: 2d layout

set property of grid container
display: grid || grid-inline
    - 

Grid container and grid items
- row axis horizontal, colum axis vertical direction
- grid lines: horizontal and vertical lines that divide grid
- grid gutter (gap): gap between rows and cols
- grid track
    - horizonal grid track: row
    - vertical grid track: col
- grid cell: single unit of css grid
- grid area: any area surrounded by 4 grid lines (can contain many gid cells)

Open with firefox and inspect to see grid lines 

fractional unit (fr)
- special measurement created for css grid
- helps us divide fractions in availiable space
